Install

Three commands. Takes about 30 seconds.

# grab the repo
git clone https://github.com/mtsatryan/openclaw-ai-agents.git

# copy skills into OpenClaw
cp -r openclaw-ai-agents/*/ ~/.openclaw/skills/

# restart and check
openclaw gateway restart

How this was built

We originally built these as paid agents for agent-hub.shop — a marketplace for Claude Code, Cline, Aider, and Codex CLI. 224 agents, tested across real client projects over 6 months.

Then we converted all of them to OpenClaw's skill format and decided to release them for free. The conversion was done by OpenClaw itself — it wrote the audit script, the conversion script, and ran both. 188 agents converted in one session, zero errors.

Each skill is 50-500 lines of curated instructions. Not a one-line prompt. Not auto-generated. Real patterns from real projects.

Works with any LLM provider you've configured in OpenClaw: Claude, GPT, DeepSeek, Gemini, or local models via Ollama.
