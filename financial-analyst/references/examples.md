# Financial Analyst — Code Examples

## Example 1

```markdown
## Company Valuation Analysis

**Company:** [Name]
**Date:** [Date]
**Purpose:** [Fundraising / M&A / Internal]

### Valuation Methods

#### 1. Revenue Multiple

| Metric | Value |
|--------|-------|
| ARR | $5M |
| Industry Multiple | 8-12x |
| Our Multiple | 10x |
| **Valuation** | **$50M** |

#### 2. Comparable Companies

| Company | Revenue | Valuation | Multiple |
|---------|---------|-----------|----------|
| Comp A | $4M | $48M | 12x |
| Comp B | $6M | $54M | 9x |
| Comp C | $5M | $45M | 9x |
| **Average** | - | - | **10x** |

#### 3. DCF Analysis

| Year | FCF | PV |
|------|-----|-----|
| Y1 | $500K | $446K |
| Y2 | $1.2M | $956K |
| Y3 | $2.5M | $1.78M |
| Y4 | $4M | $2.54M |
| Y5 | $6M | $3.41M |
| Terminal | - | $38.0M |
| **Total PV** | - | **$47.1M** |

### Valuation Summary

| Method | Value | Weight | Weighted |
|--------|-------|--------|----------|
| Revenue Multiple | $50M | 40% | $20M |
| Comparables | $50M | 35% | $17.5M |
| DCF | $47M | 25% | $11.75M |
| **Final Valuation** | - | - | **$49.25M** |

**Valuation Range:** $45M - $55M
```
